### 파일관리 단축키

ctrl + a : 전체 선택 (all)
ctrl + c : 복사 (copy)
ctrl + v : 붙어넣기 (verbatim)

컴퓨터 역사상 가장 중요한 단축키이자 가장 중요한 기능들이다. OS나 프로그램 종류와 상관없이 이러한 기능을 제공하지 않는다면 컴퓨터를 사용하는 의미가 있을까?  
리눅스에선 보안상의 이유로 위와 같은 단축키를 그대로 집어넣지 못하게 한다.  
물론 Windwos에서 제공하는 powershell이나 cmd에선 사용이 가능하다.  
리눅스는 보안 때문에 안된다던데 Windows는 허용하는 이유가 뭘까? 
근데 사실 특정 상황에서 ctrl + c는 copy가 아니라 cancel의 의미가 된다. 그래서 이러한 현상이 발생한다. (명령어의 약자 full name을 아는게 중요한 이유~)  

=> 비트들을 컴퓨터가 사용하는 주소들(url, ip adress, 하드디스크의 주소들 - https://blog.naver.com/sjhmc9695/221483714258)을 기반하여 다른 곳으로 옮기는 것이다.  
=> 무조건 인터넷상의 대량의 데이터들을 아무 생각없이 복사 하는 것보다 일정한 기준에 의해서 복사하는 것이 중요하다고 생각한다.  
=> 가치 있는 정보를 찾아내는 과정을 검색이라고 한다면 검색을 통해 얻은 데이터들을 어떻게 복사하여 더 좋은 데이터로 가공하고 검색시 더 좋은 데이터를 제공하는 것이 검색엔진을 취급하는 사람들의 역할이다.  
=> 물론 이러한 과정속에서 얼마만큼의 비용과 이익을 얻을 것인가는 항상 논란이 되는 부분이다. (즉, 돈이 될 것같으니까 개발자하는 거고 돈이 안될 것 같으니까 일반인인거임~)   

** 가치 있는 데이터, 수요가 있는 데이터를 어떻게 만들고 어떻게 가공하고 어떻게 유통시키며 어떻게 소모 되는지 이해하는 것이 매우 중요하다.  

- 사람들이 언제 컴퓨터를 사용하는가?  
- 사람들이 언제 스마트폰을 사용하는가?  
- 컴퓨터를 사용한다면 무엇 때문에 컴퓨터를 하는가?  
- 스마트폰을 사용한다면 무엇 때문에 컴퓨터를 하는가?  

=> 위 4가지 질문에 파생된 질문들에 대한 답변들이 사실상 개발자들이 컴퓨터의 언어로 다루어야 하는 영역이다.  

ctrl + insert = ctrl + c
shift + insert = ctrl + v

보통 shift는 반대 방향을 의미한다. alt + tap으로 화면을 전환할 때 alt + shift + tap을 누르면 반대 방향으로 전환이 된다.  

+ 복사의 또 다른 의미 백업

git을 이용해 commit을 사용하는 것은 매우 유용하다.  
굳이 구글 드라이브를 쓰지 않아도 git을 이용하면 더 쉽게 관리가 되는 측면이 있다.  
윈도우에서는 백업을 하려면 방대한 데이터를 시점에 따라서 저장해야 하지만 깃은 그럴 필요가 없다.  
만약 무언가를 설치하고 다시 과거로 돌리기 위해 윈도우 백업이나 클라우드 백업을 사용한다면 매우 많은 시간이 소모될 것이다.   

각 프로그램의 업데이트 시기에 따라서 미리 자동으로 이전 시점의 시스템으로 돌아가기 위해 백업을 한다면 알 수 없는 문제가 생겼을 때 정상적인 시점으로 돌아갈 수 있을 것이다.  
물론 중요한 데이터들의 경우는 따로 보관할 필요가 있다. 평상시에는 접근하지 못하게 하고 필요할 때만 요청하여 인증을 하도록 로직을 구성하는 것도 매우 중요하다.  

+ 복사의 또 다른 의미 수집

현재에도 자주 사용되는 것 중 하나는 매크로나 스크립퍼를 이용하여 인터넷 상의 방대한 데이터를 수집해 새로운 데이터를 만드는 것이다.  
뉴스 기사나 유튜브 영상등을 모두 다운로드 하여 해당 데이터를 분석하여 유의미한 정보를 만들거나 활용하는 것은 가능하다.  
한 나라의 sns를 하는 사람들의 유명한 영상들을 그대로 카피하여 쇼츠로 올리거나 딥페이크를 사용하여 없는 영상을 만들어내기도 한다.  
또한 다수의 사람들과 영상을 조작하여 매우 희귀한 상황을 연출하거나 편집을 통해 특수 효과를 교묘하게 넣어 진짜같은 가짜를 만들어낸다.  

수집하는 것보다 가공하는 것이 더 어려운 일이며 이러한 유저들을 도덕적으로 비난할 순 있지만 법적으로나 현실적으로 멈추게 할 수는 없다.
